  **Test question generator (src/lib/test-questions.ts):**
  Pure functions (no Firebase) for generating randomized test questions:
  - `generateIntervalQuestions(count)`: "What interval is C to E?" → "Major 3rd". Random root notes, random intervals. Options: 4 multiple-choice.
  - `generateFretboardQuestions(count)`: "What note is at fret 5 on the A string?" → "D". Random string + fret combos.
  - `generateScaleQuestions(count)`: "Which notes are in C major scale?" → multiple choice with correct/incorrect sets.
  - `generateChordQuestions(count)`: "What notes make up a Dm chord?" → "D, F, A". Random chord types.
  - Each returns array of { question, options, correctAnswer, category, difficulty }
  - Add Question type to types/index.ts:
    ```typescript
    interface TestQuestion { id: string; question: string; options: string[]; correctAnswer: string; category: string; difficulty: number; }
    ```

  **useAudioGenerator hook (src/hooks/use-audio-generator.ts):**
  "use client" — Web Audio API tone generation for ear training:
  - `playNote(frequency, duration)`: Play a sine wave at given frequency for given duration (ms). Use OscillatorNode + GainNode with envelope (attack 10ms, release 50ms to avoid click).
  - `playInterval(rootFreq, intervalSemitones)`: Play root note, pause 500ms, play interval note. Calculate interval frequency: `rootFreq * 2^(semitones/12)`.
  - `playChord(frequencies[])`: Play multiple frequencies simultaneously for chord identification.
  - Note frequency lookup: A4=440Hz, calculate others via `440 * 2^((midiNote - 69) / 12)`.
  - AudioContext created on first use (user gesture requirement).

  **Test service (src/lib/test-service.ts):**
  "use client" — Firestore operations:
  - `saveTestAttempt(userId, attempt)`: Save to `users/{userId}/testAttempts/{id}` with: testId, testTitle, levelId, score, totalQuestions, errors[], completedAt
  - `getTestAttempts(userId, testId?)`: Query attempts, optionally filtered by test
  - `updateProgressFromTest(userId, skillType, score)`: Update user.progressSummary for the relevant skill. Calculate weighted average with previous score.

  **useTest hook (src/hooks/use-test.ts):**
  "use client" — test state machine:
  - State: `{ mode, questions, currentIndex, answers, score, errors }`
  - Modes: "loading" → "in-progress" → "complete"
  - `startTest(type, count)`: Generate questions, enter in-progress mode
  - `answerQuestion(answer)`: Record answer, advance to next
  - `completeTest()`: Calculate score, save attempt, update progress
  - Returns current question, progress (3 of 10), score
  </action>
