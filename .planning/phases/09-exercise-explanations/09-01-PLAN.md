**1. Extend `src/lib/dig-deeper-service.ts` with exercise explanation support:**

Add a new request type alongside the existing `DigDeeperRequest`:

```typescript
export interface ExerciseExplanationRequest {
  exerciseTitle: string;      // e.g., "Identify Higher/Lower Pitch"
  exerciseType: string;       // e.g., "ear", "theory", "technique"
  question: string;           // What the exercise asked
  studentAnswer: string;      // What the student answered
  correctAnswer: string;      // The correct answer
  isCorrect: boolean;         // Whether the student got it right
  lessonTitle: string;        // Current lesson context
  levelTitle: string;         // Current level context
  levelOrder: number;         // For depth adjustment
}
```

Add a new function `streamExerciseExplanation(request: ExerciseExplanationRequest)` that:
- Builds a system prompt: "You are a music theory tutor explaining an exercise result to an electric guitar student. The student just completed an exercise in their lesson '{lessonTitle}'. Be encouraging, concise, and connect the explanation to practical guitar playing."
- Adjusts depth based on `levelOrder` (same pattern as existing `buildSystemPrompt`)
- Builds a user prompt that varies based on `isCorrect`:
  - If correct: "The student correctly answered '{question}' with '{studentAnswer}'. Briefly explain why this is correct and what concept it demonstrates. Keep it to 1-2 short paragraphs."
  - If wrong: "The student answered '{question}' with '{studentAnswer}', but the correct answer is '{correctAnswer}'. Explain why the correct answer is right, what the student might have confused, and give a tip to remember it next time. Keep it to 2-3 short paragraphs."
- Calls `client.messages.stream()` with `max_tokens: 512` (shorter than dig-deeper since these are focused explanations)
- Uses the same Claude Sonnet model as `streamDigDeeperExplanation`

**Important:** Don't modify the existing `DigDeeperRequest` or `streamDigDeeperExplanation` — add alongside. Export both types and both functions.
