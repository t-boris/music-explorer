  <name>Task 1: Create practice session Firestore service</name>
  <files>src/lib/practice-service.ts, src/hooks/use-practice-sessions.ts</files>
  <action>
  Create src/lib/practice-service.ts ("use client" — all Firestore client operations):
  - `createPracticeSession(userId, data)`: Add doc to `users/{userId}/practiceSessions` with: date (ISO string), durationMinutes, notes, exerciseIds (array of exercise IDs practiced), levelId, createdAt (serverTimestamp)
  - `getPracticeSessions(userId)`: Query `users/{userId}/practiceSessions` ordered by createdAt desc, limit 20
  - `getPracticeSession(userId, sessionId)`: Get single session doc
  - `updatePracticeSession(userId, sessionId, data)`: Update session fields
  - `deletePracticeSession(userId, sessionId)`: Delete session doc

  Use modular Firebase SDK imports (collection, addDoc, query, orderBy, limit, getDocs, doc, getDoc, updateDoc, deleteDoc, serverTimestamp).
  Use getFirebaseDb() lazy getter from firebase.ts (NOT the direct `db` export, which is null during SSR).

  Create src/hooks/use-practice-sessions.ts ("use client"):
  - `usePracticeSessions(userId)`: Custom hook using onSnapshot for real-time session list. Returns { sessions, loading, error }.
  - Uses the collection path `users/${userId}/practiceSessions` with orderBy createdAt desc.
  - Maps Firestore docs to PracticeSession type from types/index.ts.
  </action>
  <verify>TypeScript compiles with no errors. Service functions are properly typed.</verify>
