**3. Create useKnowledgeCheck hook** at `src/hooks/use-knowledge-check.ts`:
Simple state machine: "ready" → "in-progress" → "complete"
- `startCheck()`: shuffles and presents questions
- `answerQuestion(answer: string)`: checks answer, records result, advances
- `score`, `total`, `currentQuestion`, `currentIndex`
- `passed`: score >= 80% (4/5)
- `reset()`: restart
- No Firestore persistence needed — lightweight inline quiz
